📋 4 PASSI PRINCIPALI:

1️⃣  PREPARA GITHUB
   • Crea account su github.com (se non ce l'hai)
   • Crea nuovo repository (pubblico)
   • Copia l'URL del repository

2️⃣  CARICA CODICE SU GITHUB
   
   OPZIONE A - Automatico (Consigliato):
   • Apri cmd nella cartella del progetto
   • Esegui: deploy_to_github.bat
   • Segui le istruzioni
   
   OPZIONE B - Manuale:
   • cd "c:\Users\basil\Desktop\Progetto ST"
   • git init
   • git add .
   • git commit -m "Initial commit"
   • git remote add origin https://github.com/TUO-USERNAME/TUO-REPO.git
   • git branch -M main
   • git push -u origin main

3️⃣  REGISTRATI SU STREAMLIT CLOUD
   • Vai su: https://share.streamlit.io/signup
   • Registrati con il tuo account GitHub

4️⃣  DEPLOY APP
   • Accedi a: https://share.streamlit.io/
   • Clicca "New app"
   • Seleziona:
     - Repository: TUO-USERNAME/TUO-REPO
     - Branch: main
     - Main file path: app.py
   • Clicca "Deploy!"
   • Aspetta 2-3 minuti

✅ FATTO! La tua app sarà online su: https://tua-app.streamlit.app
