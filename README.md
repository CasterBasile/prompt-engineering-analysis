# Prompt Engineering Analysis Tool

Tool web per la validazione manuale sistematica delle tecniche di prompt engineering in ambito Software Testing.

## Accesso

🌐 **https://mptsanalysis.streamlit.app/**

## Tecnologie

- Python 3.x
- Streamlit
- Pandas
- Plotly

## Installazione Locale

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Utilizzo

1. Carica il file Excel con i prompt degli studenti
2. Valida manualmente i casi dubbi
3. Esporta i risultati validati
